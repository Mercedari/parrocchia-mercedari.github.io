📱 Installa su iPhone / iPad

1
Apri il file index.html in Safari (non Chrome o Firefox)
2
Tocca il pulsante Condividi ⬆ in basso nella barra di Safari
3
Scorri verso il basso e tocca "Aggiungi a schermata Home"
4
Tocca "Aggiungi" in alto a destra
5
L'icona "Parrocchia" appare sulla schermata Home — aprila come un'app!
⚠️ Deve essere aperto con Safari. Se usi Chrome o Firefox, il pulsante "Aggiungi a schermata Home" non appare.

💻 Come aprire index.html in Safari

1
Trasferisci la cartella parrocchia_pwa su iPhone (AirDrop, cavo USB o salva i file in iCloud Drive)
2
Apri l'app File sul telefono
3
Trova il file index.html → tieni premuto → "Apri con" → Safari
💡 Il modo più semplice: copia la cartella su iCloud Drive dal Mac/PC, poi aprila dall'app File su iPhone.

🤖 Installa su Android

1
Apri il file index.html in Chrome
2
Tocca i tre puntini ⋮ in alto a destra
3
Tocca "Aggiungi a schermata Home" o "Installa app"

💾 I dati sono al sicuro?

✓
I dati vengono salvati automaticamente nel browser del telefono
✓
L'app funziona anche senza connessione internet
✓
Per trasferire dati tra dispositivi: usa Esporta → Backup JSON
